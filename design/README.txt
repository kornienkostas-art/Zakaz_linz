UssurochkiRF — Offline UI Kit and Screens (HTML → PDF)

Состав:
- ui_kit_offline.html — дизайн‑токены и компоненты (кнопки, инпуты, селекты/combobox, бейджи статусов, таблицы).
- screens_offline.html — основные экраны приложения: Главное, Заказ МКЛ, Заказ Меридиан, Настройки.
- icons/ — SVG‑иконки (outline), рекомендуемые размеры 18×18 или 20×20.

Как открыть:
1) Дважды кликните на .html файлах, чтобы открыть в браузере (Chrome/Edge/Firefox).
2) Чтобы получить PDF:
   - Нажмите Ctrl+P (Печать) → Принтер: Сохранить как PDF
   - Поля: без полей/минимальные, Масштаб: 100% или Подогнать по ширине
   - Фоновые рисунки (Background graphics) — включить, чтобы сохранить цвета и тени.
   - Сохраните файл (например, UI_Kit.pdf, App_Screens.pdf)

Рекомендации по внедрению в Tkinter/ttk:
- Цвета: используйте токены из секции “Цвета”. Акцент по умолчанию — #2563EB (синий).
- Шрифты: Inter/Segoe UI, 14px база; заголовки 20–24px жирные.
- Кнопки: высота ~40–44px, радиус 10px, отступы 10–16px.
- Таблицы: высота строки 32–36px; заголовки жирные; “ЕСТЬ/НЕТ” — бейджи.
- Тени и градиенты в ttkbootstrap воспроизводите через тему (например, “flatly”), остальное — через padding/rowheight.

Примечание:
- Иконки в icons/ можно использовать напрямую (SVG → вставка в HTML) или экспортировать в PNG для Tkinter.
- Если нужна единая PDF‑книга — откройте оба HTML и сохраните в два PDF, затем объедините любым PDF‑инструментом.
