Поместите сюда логотип приложения для окна и системного трея.

Рекомендуемое имя файла:
- logo.png

Рекомендуемое разрешение:
- 128x128 для трея (будет автоматически изменён при необходимости)
- PNG с прозрачным фоном

Путь к логотипу по умолчанию в настройках:
- "tray_logo_path": "app/assets/logo.png"

Если файл отсутствует, приложение использует встроенную простую иконку.
